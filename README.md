# Telegram Бот для Репетитора

Бот для управления расписанием уроков, напоминаний и проверки домашних заданий.

## Возможности

### Для учеников:
- Регистрация через /start с выбором часового пояса
- Просмотр своего расписания
- Автоматические напоминания об уроках
- Отчет о выполнении домашнего задания

### Для репетитора:
- Управление расписанием учеников
- Назначение и изменение уроков
- Поддержка групповых занятий
- Получение отчетов по ДЗ
- Ежедневное напоминание о расписании
- Настройка времени уведомлений
- Учет часовых поясов

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Создайте файл `.env`:
```
BOT_TOKEN=ваш_токен_от_BotFather
ADMIN_ID=ваш_telegram_id
```

3. Запустите бота:
```bash
python src/main.py
```

## Получение токена

1. Найдите [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте `/newbot`
3. Следуйте инструкциям
4. Скопируйте токен

## Получение Telegram ID

1. Найдите [@userinfobot](https://t.me/userinfobot)
2. Отправьте `/start`
3. Скопируйте ваш ID

## Настройки

Настройки хранятся в `data/settings.json`:
- `admin_timezone` - часовой пояс репетитора (по умолчанию 3 - МСК)
- `reminder_hours_before` - за сколько часов напоминать (по умолчанию 1)
- `homework_check_minutes_before` - за сколько минут отчет по ДЗ (по умолчанию 5)
- `admin_daily_reminder_time` - время ежедневного напоминания (по умолчанию "08:00")

## Часовые пояса

Бот автоматически учитывает часовые пояса:
- Репетитор ведет расписание в своем времени
- Каждый ученик указывает свой часовой пояс при регистрации
- Все уведомления приходят в правильное время для каждого
